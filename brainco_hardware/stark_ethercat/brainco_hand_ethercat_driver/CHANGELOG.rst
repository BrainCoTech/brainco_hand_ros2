1.0.0 (2025-10-31)
------------------

* Initial release of brainco_hand_ethercat_driver package
* EtherCAT hardware interface implementation for Revo2 dexterous hands
* Support for both left and right hand configurations
* Integrated ros2_control with trajectory controllers
* Real-time EtherCAT communication support
* Joint state feedback (position, velocity, effort)
* MoveIt integration support
