1.0.0 (2025-10-23)
------------------

* Initial release of brainco_gazebo package
* Added complete Gazebo simulation environment for BrainCo Revo2 dexterous hands
* Support for both left and right hand configurations
* Integrated ROS2 control with trajectory controllers
* Mimic joint control for automatic finger coupling
* RViz visualization support
